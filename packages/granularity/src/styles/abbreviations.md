# Style token abbreviations

| Short | Full name |
| --- | --- |
| `bg` | `background` |
| `fg` | `foreground` |
| `brd` | `border` |

## Naming rule

- Use short forms in CSS custom properties and related style token names inside `src/styles`.
- Example mappings:
  - `--background` → `--bg`
  - `--foreground` → `--fg`
  - `--border` → `--brd`
